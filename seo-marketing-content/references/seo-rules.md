# SEO Writing Rules

## On-Page SEO Checklist

### Title Tag (SEO Title)
- 50-60 characters max (Google truncates at ~60)
- Primary keyword near the front
- Include a power word: "Ultimate", "Complete", "Best", "Proven", "Easy"
- Avoid clickbait; match search intent

### Meta Description
- 150-160 characters
- Primary keyword included naturally
- Clear value proposition
- End with a soft CTA: "Learn more", "Discover how", "Find out why"

### H1
- One per page, matches or mirrors the SEO title
- Contains primary keyword
- Should be a natural sentence, not keyword-stuffed

### Body Content – Keyword Placement Rules
| Location | Rule |
|---|---|
| First 100 words | Must contain primary keyword once |
| H2 headings | 2-3 H2s should include keyword or close variations |
| Throughout body | Natural placement, 1-2% density |
| Last paragraph | Include keyword at least once |
| Image alt text | Suggest a keyword-rich alt text |

### LSI Keywords (Latent Semantic Indexing)
- Include 5-8 semantically related terms
- These help Google understand topic breadth
- Example for "matcha latte": green tea powder, antioxidants, L-theanine, ceremonial grade, caffeine alternative

### Readability Rules
- Sentences: max 20 words average
- Paragraphs: max 3-4 sentences
- Use active voice 80%+ of the time
- Flesch reading score: aim for 60+ (plain language)
- Bullet lists for 3+ items
- One idea per paragraph

### Internal & External Links
- Suggest 1-2 internal link placements (use placeholder: [Link to: related page])
- Mention 1 external authority source (e.g., "According to Harvard Medical School...")

### Content Structure Template

```
# [H1: Primary Keyword + Benefit]

[Introduction: 100-150 words. Hook sentence. Include keyword. Preview what reader will learn.]

## [H2: What is {keyword}? / Why {keyword} Matters]
[150-200 words. Define, contextualize, cite a stat if possible.]

## [H2: Top Benefits / How to / Key Strategies for {keyword}]
[200-300 words. Core value section. Use bullet points or numbered list.]

## [H2: Common Mistakes / What to Avoid / Pro Tips]
[150-200 words. Differentiation. Practical and actionable.]

## [H2: How to Get Started / Step-by-Step Guide]
[150-200 words. Actionable takeaways.]

## Conclusion
[100-150 words. Summarize key points. Restate keyword. End with CTA.]

---
**Suggested Keywords:** [list 5-8 LSI terms]
**SEO Title:** [final title]
**Meta Description:** [150-160 char description]
```
