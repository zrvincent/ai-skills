# Content Type Format Reference

## Blog Post / Article
- Length: 800–2500 words depending on topic complexity
- Structure: Intro → Problem → Solution sections (H2s) → Conclusion + CTA
- Tone: Educational, helpful, authoritative
- CTA: Subscribe / Read more / Contact us / Download resource

## Landing Page
- Length: 300–800 words (above the fold matters most)
- Structure: Hero headline → Value proposition → Benefits (bullets) → Social proof → CTA button
- Tone: Persuasive, benefit-focused, urgency where appropriate
- SEO note: Landing pages target transactional keywords ("buy", "get", "hire", "best X for Y")
- CTA: Primary button CTA (e.g., "Get Started Free", "Book a Demo") — repeat 2–3x on page

## Product Description (E-commerce)
- Length: 150–400 words
- Structure: Hook sentence → Key features (bullets) → Benefits paragraph → Technical specs (optional) → CTA
- Tone: Conversational, benefit-driven, sensory where applicable
- SEO note: Include product-specific keywords (brand + model + use case)
- CTA: "Add to Cart" / "Shop Now" / "Order Today"

## Social Media Caption
- Length: 
  - Instagram: 138–150 chars for preview, up to 2,200 chars total
  - LinkedIn: 150 chars preview, 700 chars recommended max
  - Facebook: 40–80 chars for best engagement
- Structure: Hook → Value/Story → CTA → Hashtags (Instagram/LinkedIn)
- Tone: Platform-native (LinkedIn = professional; Instagram = lifestyle; Facebook = community)
- SEO note: Social doesn't directly affect Google SEO but hashtags act as social search keywords
- CTA: "Link in bio" / "Comment below" / "Tag someone who needs this"

## Email Newsletter / Marketing Email
- Length: 200–500 words for campaigns; up to 800 for newsletters
- Structure: Subject line → Preview text → Personalized greeting → Value body → CTA button → Footer
- Subject line: Under 50 chars, curiosity or benefit-driven, avoid spam trigger words
- Tone: Conversational, personal ("you" language), one clear message per email
- SEO note: Email isn't indexed, but drives traffic to SEO pages — include links
- CTA: One primary CTA button; optional secondary text CTA
