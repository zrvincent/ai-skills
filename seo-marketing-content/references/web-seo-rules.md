# Web SEO Content Rules

## Article Types
- **Blog Post**: Informational, 800–1500 words, conversational tone
- **Landing Page**: Persuasion-focused, 500–1000 words, benefit-driven, strong CTA
- **Pillar Page**: Comprehensive guide, 2000–4000 words, covers topic exhaustively
- **Product/Service Page**: 300–700 words, feature + benefit format, trust signals

## SEO Title Rules
- 50–60 characters max
- Primary keyword as close to start as possible
- Include power words: Best, Guide, How to, Tips, [Year]
- Example: "Organic Skincare for Sensitive Skin: 10 Expert Tips (2025)"

## Meta Description Rules
- 150–160 characters
- Include primary keyword naturally
- End with a soft CTA ("Learn more", "Discover how", "Start today")
- Unique — never duplicate across pages

## Heading Structure
```
H1: Primary keyword (once only)
  H2: Major subtopic 1 (LSI keyword)
    H3: Sub-point A
    H3: Sub-point B
  H2: Major subtopic 2
  H2: FAQ
  H2: Conclusion / CTA
```

## Introduction Formula (First 100 Words)
1. Hook — surprising stat, question, or bold statement
2. Problem — identify the reader's pain point
3. Promise — what this article will solve/teach
4. Keyword — include naturally within first 100 words

## Content Body Best Practices
- Short paragraphs (2–4 lines max)
- Use bullet points and numbered lists for scanability
- Bold key phrases (not entire sentences)
- Include transition words for readability score
- Add 1 internal link placeholder every 300–400 words
- Include one external authority reference (paraphrased, not quoted)

## FAQ Section (People Also Ask Style)
- 3–5 questions using natural language ("How do I…", "What is…", "Is it safe to…")
- Answers: 40–60 words each (concise for featured snippet potential)
- Include primary or LSI keyword in at least 2 questions

## Call to Action
- One primary CTA at end of article
- Optional secondary CTA mid-article for long posts
- Types: Subscribe, Download, Contact, Shop, Book, Learn More

## Keyword Density
- Primary keyword: ~1–1.5% of total word count
- LSI keywords: sprinkled naturally throughout
- Avoid: exact match repetition within same paragraph
